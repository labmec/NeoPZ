This group has materials related projecting a given solution into a discrete approximation space. These classes can be used to see how to interact with \(H^1(\Omega)\), \(H(\text{div},\Omega)\) and \(H(\text{curl},\Omega)\) approximation spaces.

TPZL2ProjectionÂ¶

template<class TVar = STATE>
class TPZL2Projection : public TPZMatBase<STATE, TPZMatSingleSpaceT<STATE>, TPZMatErrorSingleSpace<STATE>>Â¶

Implements an L2 projection of a given solution on a scalar approximation space. This material uses a scalar approximation space in a geometric domain of dimension defined by the user (1, 2 or 3D). It can project several solutions at once, but for calculating the error, it will only consider one solution at a time. The solutions to be projected are given by the Forcing Function of the TPZMaterialT. It is expected to be a TPZVec<TVar> of size nsol. For the error analysis, the solution is set through the ExactSol of TPZMatErrorSingleSpace.

TPZHDivProjectionÂ¶

template<class TVar = STATE>
class TPZHDivProjection : public TPZMatBase<STATE, TPZMatSingleSpaceT<STATE>, TPZMatErrorSingleSpace<STATE>>Â¶

Implements a projection of a given solution on a HDiv approximation space. This material uses a HDiv approximation space in a geometric domain of dimension defined by the user (2 or 3D) and projects a given solution using the HDiv norm. The solution to be projected is set by the Forcing Function of the TPZMaterialT. It is expected to be a TPZVec<TVar> of size 4, as the solution is always in a 3D space and the last position is the divergence. For the error analysis, the solution is set through the ExactSol of TPZMatErrorSingleSpace.

Modal Analysis of WaveguidesÂ¶

The following classes implement a \(H(\text{curl},\Omega)\times H^1(\Omega)\) formulation for the analysis of waveguides. More precisely, they implement:

Find non-trivial \(\left(\beta^2,{e_t}, {e_z}\right) \in (\mathbb{C} \times [\mathbb{C}]^N \times [\mathbb{C}]^M)\) such that:

\[\begin{split}\begin{bmatrix} A_{tt} & 0 \\ 0 & 0 \end{bmatrix} \begin{Bmatrix} e_t \\ e_z \end{Bmatrix} = - \beta ^2 \begin{bmatrix} B_{tt} & B_{tz} \\ B_{zt} & B_{zz} \end{bmatrix} \begin{Bmatrix} e_t \\ e_z \end{Bmatrix} \text{,}\end{split}\]

where

\[\begin{split}[A_{tt}]_{ij} &= \int_{\Omega} \left[ \mu_{zz}^{-1}\left( \nabla_t\times \boldsymbol{\varphi}_i \right) \cdot \left( \nabla_t\times \boldsymbol{\varphi}_j \right) - k_0^2 \boldsymbol{\epsilon}_{xy} \boldsymbol{\varphi}_i\cdot \boldsymbol{\varphi}_j\right] d\Omega\text{,}\label{eq:fem-mat-1}\\ [B_{tt}]_{ij} &= \int_{\Omega} \boldsymbol{\mu}_{xy^S}^{-1}\boldsymbol{\varphi}_i \cdot \boldsymbol{\varphi}_j\, d\Omega\text{,}\label{eq:fem-mat-2}\\ [B_{tz}]_{ij} &= \int_{\Omega} \boldsymbol{\mu}_{xy^S}^{-1}\boldsymbol{\varphi}_i \cdot \nabla_t\, \varphi_j\, d\Omega\text{,}\label{eq:fem-mat-3}\\ [B_{zt}]_{ij} &= \int_{\Omega} \boldsymbol{\mu}_{xy^S}^{-1}\nabla_t\, \varphi_i \cdot \boldsymbol{\varphi}_j \, d\Omega\text{,}\label{eq:fem-mat-4}\\ [B_{zz}]_{ij} &= \int_{\Omega} \left[ \boldsymbol{\mu}_r^{-1}\nabla_t\, \varphi_i \cdot \nabla_t\, \varphi_j - k_0^2 \epsilon_{zz} \varphi_i \varphi_j\right] d\Omega\text{.}\label{eq:fem-mat-5}\end{split}\]

This formulation was presented originally in this work 1. In this work 2 a very detailed derivation along with results obtained using NeoPZ are presented.

TPZWaveguideModalAnalysisÂ¶

This class supports materials with the following form of anisotropy:

\[\begin{split}\epsilon_r = \begin{bmatrix} \epsilon_{xx} & 0 & 0\\ 0 & \epsilon_{yy} & 0\\ 0 & 0 & \epsilon_{zz}\\ \end{bmatrix}\,\,\,\,\, \mu_r = \begin{bmatrix} \mu_{xx} & 0 & 0\\ 0 & \mu_{yy} & 0\\ 0 & 0 & \mu_{zz}\\ \end{bmatrix}\end{split}\]
class TPZWaveguideModalAnalysis : public TPZMatBase<CSTATE, TPZMatCombinedSpacesT<CSTATE>, TPZMatGeneralisedEigenVal>Â¶

This class implements the weak statement for the modal analysis of waveguides using HCurl and H1 elements. It uses a 2D Hcurl space for the transversal components of the electric field and an 1D H1 space for the longitudinal component.

class TPZDarcyFlow : public TPZMatBase<STATE, TPZMatSingleSpaceT<STATE>, TPZMatErrorSingleSpace<STATE>, TPZIsotropicPermeability>Â¶

This class implements an H1-conforming approximation for the Darcy flow equation for isotropic materials.

The Darcy flow equation is given by:

\[- \nabla \cdot K \nabla u = f,\]
where \(u\) is the pressure field to be solved, \(K\) is the permeability tensor and \(f\) is the source term.

See

TPZMixedDarcyFlow For an approximation using the mixed method.

class TPZMixedDarcyFlow : public TPZMatBase<STATE, TPZMatCombinedSpacesT<STATE>, TPZMatErrorCombinedSpaces<STATE>, TPZIsotropicPermeability>Â¶

This class implements a mixed approximation for the Darcy flow equation for isotropic materials.

The Darcy flow equation is given by:

\[\nabla \cdot \boldsymbol{\sigma} = f,\]
where \(\boldsymbol{\sigma} = -K \nabla u\) and \(u\) are the flux and pressure field to be solved respectively, \(K\) is the permeability tensor and \(f\) is the source term.
