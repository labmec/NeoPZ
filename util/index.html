TPZSimpleTimer¶

class TPZSimpleTimer¶

Simple timer Utility This timer allows to measure execution time with nested timers support. The time range corresponds to the lifetime of the TPZSimpleTimer instance. For nested timers, one can configure if the information will be printed only when the outermost timer is complete or not.
